Ajouter une vidéo comme header

Pour ajouter une vidéo comme header, c'est très simple. Celle-ci sera ajoutée juste avant le message de bienvenu.
Vous pouvez ajouter n'importe quel contenu en iframe. A noter que l'iframe se comporte comme un fond (background)

Rechercher : <body id="page-top" data-spy="scroll" data-target=".navbar-fixed-top">
